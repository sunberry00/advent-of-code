# Advent of Code Python Solutions

This repository contains my solutions for [Advent of Code](https://adventofcode.com/) in Python.

## Repository Structure

- `2024/`: Folder containing solutions for 2024, with each day's solution in a separate Python file (e.g., `day01.py`, `day02.py`, etc.).
- `input/`: Folder where the input files for each day will be stored.
- `utils/`: Utility scripts, such as `input_reader.py` to load input files.

## Running the Solutions

1. Download the input files for each day from [Advent of Code](https://adventofcode.com/).
2. Save the input file in the `input/{year}` directory. The file should be named `dayXX.txt`, where `XX` is the day number (e.g., `day01.txt` for day 1).
3. Run the solution for a specific day by executing the corresponding Python file, e.g., for 2024 day 1:

```bash
python 2024/day01.py
